# CarMax Pre-Trip Inspection Checklist
A mobile-friendly pre-trip inspection checklist for CarMax two-car hauler truck drivers.
## Features
- 12-step comprehensive inspection
- Auto-save progress
- Works on desktop and mobile
- iOS and Android compatible
## How to Use
1. Open the checklist on your phone or computer
2. Enter today's date and truck number
3. Follow each step and check items as you complete them
4. Progress saves automatically
